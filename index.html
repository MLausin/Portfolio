My Recent Project

Tic Tac Toe Game Python and Tkinter Client given Project | October 25, 2023 December 01, 2025 Developed by BitPowerType (Dagupan City) Python Tkinter

Designed and implemented a classic Tic Tac Toe game as a final project for University of Pangasinan. The project aimed to showcase proficiency in Python programming and graphical user interface (GUI) development using Tkinter. The game provides a user friendly and interactive platform for players to enjoy the timeless Tic Tac Toe experience. Features:

Created a two-player Tic Tac Toe game with a 3x3 grid, allowing players to take turns placing their marks (X or O). Designed an intuitive and visually appealing GUI using Tkinter, enhancing the overall gaming experience,

implemented game logic to determine the winner or declare a draw based on the arrangement of marks on the grid.
